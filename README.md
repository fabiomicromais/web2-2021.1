1. **[Git](https://git-scm.com/downloads)**
2. **[Docker](https://www.docker.com/)**
3. **[Laradock](laradock.io)**
4.  **[WSL 2](aka.ms/wsl2)**
5.  **[Visual Studio Code](https://code.visualstudio.com/download)** ℹ️ (Opcional)

## 🛠️ Como instalar o Carsystem
Após executar o download e instalação das ferramentas listadas acima, é necessário clonar o repositório no seu computador usando o seguinte comando:
```
git clone <link>
```


Depois de instalar o `Docker` e o `laradock` entre na pasta criada e use o seguinte comando: 
> 
 ```
 docker-compose up -d mysql phpmyadmin apache2 workspace
 ```
***Esse comando pode levar alguns minutos dependendo da configuração do teu hardware.***


  Em seguida entre na pasta clonada pelo gerenciador de arquivos do seu computador ou utilizando o seguinte comando:
```
cd + Tecla tab ou cd web2-2021.1
```

*Feito isso, o sistema estará pronto pra você usá-lo em sua máquina!* 🌟

---

### 📌 Funcionalidades do sistema:
  - Cadastro de funcionários, clientes, fornecedores.
  - Atualização dos dados de funcionários, clientes e fornecedores.
  - Exclusão de fornecedores, clientes e funcionários.

### 😧 Funcionalidades não implementadas:
  - Compras de veículos
  - Vendas de veículos
  - Login e logout
  - Criação de contas
  - Pesquisa de dados
  - Upload de imagens
